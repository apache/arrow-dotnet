Class ArrowBuffer.Builder<T>

Namespace
Apache.Arrow
Assembly
Apache.Arrow.dll

The ArrowBuffer.Builder<T> class is able to append value-type items, with fluent-style methods, to build up an ArrowBuffer of contiguous items.
